# Nicolas Reg Collection

This is a small collection of multiple Windows Registry files.

# What it includes

- **```Enable Estimated Battery Time.reg```** :
    - Enables the battery life estimated time on devices where this is disabled

- **```EnableLongPath.reg```** :
    - Enables Windows Explorer's ability to handle long paths.

- **```Fix ICS being sussy on reboot.reg```** :
    - Fixes Windows's ICS (Internet Connection Sharing) disabling itself every time you reboot, sleep or hibernate your PC.

- **```No Bingus in Windows Search.reg```** :
    - Disables web/Bing searches in Windows Search.

# How to use

Download the desired reg file from the source code and execute the file.  
Note : some of them requires a reboot to be effective.
